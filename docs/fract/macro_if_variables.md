# Macro if variables

+ ``OS`` <br>
  Operating system name. <br>
  Possible values: ``darwin``, ``linux``, ``windows``.

 + ``ARC`` <br>
  Architecture. <br>
  Possible values: ``386``, ``amd64``, ``amd64p32``, ``arm``, ``arm64``, ``ppc64``, ``ppc64le``, ``mips``, ``mipsle``, ``mips64``, ``mips64le``, ``mips64p32``, ``mips64p32le``, ``ppc``, ``s390``, ``s390x``, ``sparc``, ``sparc64``.
